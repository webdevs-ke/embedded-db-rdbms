To create the Demo App database, RUN each of these SQL Commands
separately.

  1. CREATE DATABASE library
  2. SHOW DATABASES [optional]
  3. USE library
  4. SHOW TABLES [optional]
  5. CREATE TABLE users (id INT PRIMARY, name TEXT, email TEXT UNIQUE,
    phone TEXT UNIQUE)
  6. CREATE TABLE book_types (id INT PRIMARY, type TEXT UNIQUE)
  7. INSERT INTO book_types VALUES (1, Fiction)
  8. INSERT INTO book_types VALUES (2, Non-Fiction)
  9. CREATE TABLE books (id INT PRIMARY, title TEXT UNIQUE, file BLOB,
    typeID INT, userID INT)

Once you're done, open the demo App, fill in the register form to create
your account. You will use this info to login to your library profile.


NB: Although it is possible to INSERT your profile details using the SQL Console,
it is not advised since your data has to undergo validity constraint checks
before the Database & Persistence engines can accept and store it.


If you prefer, you can automate the database creation process just by
opening the demo App and entering your registration info which triggers the
creation of the library database if it doesn't already exist.
